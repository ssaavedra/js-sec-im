    Tools

                                                           Titaniumcore Project
================================================================================
Atsushi Oka [ <a href="http://oka.nu/">http://oka.nu/</a> ]                                       Jan 23,2009

This directory contains following scripts.

    <a href="./packages.js">packages.js</a>
        This is a simple framework to manages dependency among the scripts.
        This is probably the most important utility for this library.
        See <a href="../tools/packages.readme.txt">packages.readme.txt</a>

    <a href="./binary.js">binary.js</a>
        Implements base64/utf-8 binary string conversion.
        See <a href="../tools/binary.readme.txt">binary.readme.txt</a>

    <a href="./elapse.js">elapse.js</a>
        A simple performance profiler.
        See <a href="../tools/elapse.readme.txt">elapse.readme.txt</a>

    <a href="./isarray.js">isarray.js</a>
        Array.prototype.isArray=true;
        See <a href="../tools/isarray.readme.txt">isarray.readme.txt</a>

    <a href="./trace.js">trace.js</a>
        Print debug messages.
        See <a href="../tools/trace.readme.txt">trace.readme.txt</a>

    <a href="./trace.async.js">trace.async.js</a>
        Print debug messages with asynchronous processing feature enabled.
        See <a href="../tools/trace.async.readme.txt">trace.async.readme.txt</a>
